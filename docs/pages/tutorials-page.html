Introduction

This guide will help walk you through a basic footwork.js based application.

viewModels

Creating objects (viewModels) to bind to the DOM is done exactly like it is in knockout.js:

// Works just like regular knockout.js code
var viewModelObject = {
  prop: 'value'
};
fw.applyBindings(viewModelObject); // this works as you would expect

To get the additional features that footwork provides, such as namespacing, lifecycle events, component integration, and routing...then you need to create them via the fw.viewModel() based class constructors.
